# Restaurant-Inspection-Analytics-Dashboard-Power-BI-Microsoft-Fabric
Built an end-to-end business intelligence solution to monitor restaurant inspection performance, sanitation grades, violations, and regional trends using Microsoft Fabric and Power BI.

This project demonstrates real-world skills in:

Data Cleaning & Transformation
- Data Modeling
- Row-Level Security (RLS)
- Scheduled Refresh Automation
- Gateway Configuration
- Interactive Dashboard Development
- Cloud Deployment using Microsoft Fabric
- 
## Tech Stack
Microsoft Fabric
Power BI
Genflow (Data Cleaning / ETL)
Data Gateway
DAX

RLS Security
Scheduled Refresh
